# QuizMakerWPF

A WPF application for creating quizes.


## Features

- Clean, easy to use UI
- Saving quizes as .db SQLite database file
- Multiple quizes per file
- User can modify existing Quizes database and edit/add/delete questions


## Tech Stack

C#, WPF, SQLite
